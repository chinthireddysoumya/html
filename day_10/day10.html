SUMMARY OF HTML

    HTML:

  • It is a standard markup language used to structure web page and its content
  • It contains series of elements
  • It tells the browser how to display the content

    HTML tags:

  • pair tags-consists of two instructions an opening tag and closing tag
  • self closing-It is a tag that does not need to be closed manually,which means it does not have a separate closing tag

    HTML attributes:

  • Attributes provide additional information about elements
  • They are always specified in the starting tag
  • Attributes usually come in name/value pair
  • Attributes are:href,title,target

    HTML tables:

  • HTML tabkes allows developers to arrange data into rows and columns
  • It consists of cells inside rows and columns
  • Table consists of tr-table row,th-table heading,td-table data

    HTML images:

  • It is defined with img tag
  • It is used ti embed image into the document
  • It consists of attribures such as-
    • =>src-for defining the path or url of teh image
    =>title-shows the details about the image when hovered
    =>alt-used to shoe alternative text when image is broken

    HTML Formating tags:

  • These are self closing tags
  • b-bold text
  • i-italic text
  • strong-important text
  • mark-marked text
  • del-deleted text
  • sub-subscripted text
  • sup-superscript
